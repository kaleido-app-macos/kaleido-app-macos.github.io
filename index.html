Installation

1

Download the DMG file

Download Kaleido Pro v1.0 (67 MB)

2

Open the DMG and drag Kaleido Pro to Applications

Double-click the downloaded .dmg file and drag Kaleido Pro.app into your Applications folder.

3

First launch — allow the app in macOS Security settings

macOS blocks all apps downloaded from the internet on first launch. When you see the warning:

  1. Click Done on the warning dialog
  2. Open System Settings > Privacy & Security
  3. Scroll down to the Security section
  4. Find the message "Kaleido Pro.app was blocked" and click Open Anyway
  5. Enter your password when prompted

This is a one-time step. After this, the app opens normally. Kaleido Pro is signed and notarized by Apple.

4

Enjoy!
